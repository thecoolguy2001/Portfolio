March 2023

Visual Design Direction

An extension popup is a tiny canvas with high stakes — the user has to understand controls instantly. The visual language was tuned for a 320-pixel surface that lives one click below the address bar.

Status before settings

The popup leads with whether protection is on for the current site. Settings sit underneath — most users only check the toggle.

Blur as the brand

The product's identity is the blur itself. The icon, the highlight color, the popup all echo a soft frosted gradient.

Plain language only

No "DOM scanning" or "regex match" jargon. Copy speaks to what the user sees: "this word is blurred," "show it anyway."

Per-site memory

Toggles remember per origin so power users aren't re-configuring on every domain.

March 2023

The Solution

A lightweight content script that scans page text, matches against a customizable word list, and applies CSS blur effects to just the offending terms.

Targeted Filtering

Only blurs specific words, preserving full website functionality and context.

Customizable Lists

Users can add or remove words from the filter list based on personal preferences.

Lightweight Design

Minimal performance impact ensures browsing remains fast and responsive.

February 2023

Target Users

Two user types ground every design choice — a parent protecting younger eyes, and an adult choosing a calmer browser for themselves.

PG

The Protective Parent

Wants a soft layer of safety on the family laptop

Goals

  • Block specific harmful or distressing words
  • Keep websites usable, not nuked
  • Configure once, forget about it

Pain Points

  • Whole-site blockers are too blunt
  • Most parental tools cost a subscription
CA

The Curating Adult

Wants a calmer feed with less inflammatory language

Goals

  • Filter slurs, spoilers, or trigger words
  • Decide what to soften — not have it dictated
  • Toggle off for specific sites instantly

Pain Points

  • Generic safe-search tools are inflexible
  • Manual content avoidance is exhausting

February 2023

The Problem

Users frequently encounter distressing or inappropriate language online, but blanket content blockers can break site layouts or hide too much information.

There was a need for a more surgical approach that could filter specific harmful words while maintaining the overall browsing experience.

February 2023

Project Started

Safe Browsing+ is a Chrome extension designed to enhance online safety by selectively blurring harmful or offensive words on web pages.

Combining an intuitive UI with powerful background scripting, it preserves full website functionality while giving users control over their browsing experience.
